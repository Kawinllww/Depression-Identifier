# Depression-Identifier
## จุดประสงค์:

-เพื่อศึกษาปัจจัยสำคัญที่ทำให้เกิดโรคซึมเศร้า ในช่วงอายุต่างๆ

-สร้างเครื่องมือที่สามารถตรวจโรคซึมเศร้าได้อย่างแม่นยำ

-เพื่อแก้ปัญหาการที่คนคิดไปเองว่าตัวเองเป็นซึมเศร้า

-เพื่อจำแนกอายุที่เป็นโรคซึมเศร้ามากที่สุด
## สถานการณ์

-ณ ตอนนี้มีคนไทยจำนวนมากที่สงสัยว่าตัวเองเป็นโรคซึมเศร้า ดังนั้น เครื่องมือนี้จะเป็นประโยชน์อย่างมาก

-โรคซึมเศร้าตอนนี้ จัดเป็นโรคที่ทำให้มีคนเสียชีวิตจากการฆ่าตัวตาย
## Data Mining

-เป้าหมายทางธุรกิจ: เพื่อเพิ่มประสิทธิภาพในการวินิจฉัยโรครวมถึงลดระยะเวลาและค่าใช้จ่ายด้านสุขภาพในระยะยาว

-เป้าหมายเชิงเทคนิค: การพัฒนาโมเดล AI ที่มีความแม่นยำสูงและมีค่าความแม่นยำ (accuracy) อย่างน้อย 80%, สร้างกลไกที่สามารถตรวจจับปฏิสัมพันธ์ระหว่างตัวแปรที่ซับซ้อน ได้โดยใช้เทคนิค feature engineering ที่เหมาะสมเพื่อค้นหารูปแบบที่ซ่อนอยู่

## แผนดำเนินการ
-ทำการสำรวจข้อมูลของคน 500 คน ในช่วงอายุ 18-34 ปี แบบสุ่ม โดยติดต่อหน่วยงานที่เกี่ยวข้องกับโรคซึมเศร้า

-สร้างตัวบ่งชี้ภาวะโรคซึมเศร้า โดยใช้ AI

-หากตัวบ่งชี้มีความแม่นยำมากกว่า 80% จะนำตัวบ่งชี้ไปใช้จริง
